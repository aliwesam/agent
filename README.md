# Football Player Voting App

This simple Flask application allows users to vote for the best football player of the year. Votes are stored in memory and results can be viewed after submitting a vote.

## Setup

1. Install dependencies:
   ```bash
   pip install flask
   ```
2. Run the app:
   ```bash
   python app.py
   ```
3. Visit `http://localhost:5000` in your browser to vote.
